# Problema 1.
 Elabore um programa multithread que recebe `dez valores inteiros para um vetor vA` 
e outros  `dez  valores  inteiros  para  um  vetor  vB` 
* (para  receber  os  valores  utilize  apenas  a  thread principal).

O programa deve fazer a soma destes dois vetores utilizando `quatro threads` gerando um vetor resultante Vc. Depois reescreva o seu programa para receber `mil valores por vetor ao invés de apenas dez.`

##### Lembrete:

```sh
vC[0] = vA[0] + vB[0], ..., vC[n] = vA[n] + vB[n]
```
